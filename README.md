# BluemapCMI Integration Plugin

A Minecraft plugin that seamlessly integrates **BlueMap** and **CMI** plugins, displaying important locations like spawn and warps on the map.
This is a addon for a Bluemap plugin, to make this plugin works without problems you must have downloaded both **Bluemap** and **CMI** plugins.
